## Tech Stack

### Frontend
- React + Vite

- Axios for API requests
- Google AI for diet recommendations

### Backend
- Node.js + Express
- MongoDB with Mongoose
- JWT for authentication
- Bcrypt for password hashing
- Zod for validation
